# Doxa

## Introduction
Doxa is an AI Agent focused on the extraction and analysis of analysis of complex technical-legal documentation from public auction portals and tender platforms.


https://github.com/user-attachments/assets/6443ce70-119b-43b7-8593-674e28cdf6b4


## How does it work?
1. Copy the URL of the public auction porta (as per now, only astegiudiziarie.it is supported)
2. Let Doxa download the full documentation and train on it
3. Ask Doxa whatever you need.
